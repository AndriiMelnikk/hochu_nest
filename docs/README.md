## Основні концепції

### Зворотний аукціон

Платформа працює за принципом зворотного аукціону:
- Покупці створюють запити на послуги
- Продавці надсилають пропозиції
- Покупці обирають найкращу пропозицію

### Ролі користувачів

- **Buyer** (покупець) - створює запити, приймає пропозиції
- **Seller** (продавець) - надсилає пропозиції
- **Admin** (адміністратор) - модерує контент, управляє платформою

### Гейміфікація

Система включає:
- XP (досвід) за різні дії
- Рівні для покупців та продавців
- Досягнення, які розблоковуються автоматично

## Технології

- **Backend:** NestJS
- **Database:** MySQL/MariaDB
- **Authentication:** JWT
- **Real-time:** WebSocket (Socket.io)
- **File Storage:** Cloud Storage (AWS S3, Cloudinary)
